# PRX HUD

- Only ESX (By the moment)
- Street Display at vehicle
- Status Bars:
  - Food
  - Drink
  - Radio (pma-voice / rp-radio)
  - Voice (pma-voice)
 
- Speedometer
  - Fuel Indicator
  - Damage Indicator
  - Speed limiter
  - Seatbelt

- Low resmon:
  - At Vehicle 0.03ms
  - Whitout vehicle 0.00ms to 0.01ms max
